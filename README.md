# 🦐 海鲜捞汁记账

一个给海鲜摊主用的**单文件记账 Web APP**。零依赖、离线可用、手机电脑都行，
专为「全场 20 元/半斤」的捞汁小海鲜摊设计。

**在线使用：<https://phyphy-quant.github.io/seafood/>**

## 功能

- 🧾 **卖货**：点一下品类就记一笔（默认 1 份 = 半斤 = 20 元），份数、价格可改，支持补记往日
- 📦 **进货支出**：快捷按钮一键带入常用品类与批发单价，只需填数量；单位下拉可选（斤/件/袋/盒…）也可自定义；摊位费、燃气、打包盒等支出分类记录
- 📊 **报表**：今日/本月收入、支出、净利，品类销量排行，近 7 天走势，每日汇总表
- 💾 **CSV 导出/导入**：流水、每日汇总、进货记录三种表格，Numbers / Excel / WPS 直接打开
- 📴 **完全离线**：没网也能记账，不需要注册登录

## 📱 iPhone 安装成 APP

1. 用 **Safari** 打开 <https://phyphy-quant.github.io/seafood/>（一定要 Safari，不能在微信里打开）；
2. 点底部中间的**分享按钮**（方框带向上箭头 ⬆️）；
3. 往下滑，点「**添加到主屏幕**」；
4. 名称保持「海鲜记账」，点右上角「**添加**」。

完成后主屏幕会出现 🦐 图标，点开就是全屏 APP，**之后没网也能用**。

## 💻 Mac 安装到程序坞（Dock）

用 Safari 打开同一个网址 → 菜单栏 **文件 → 添加到程序坞…** → Dock 出现 🦐 图标。

## ⚠️ 重要：数据存在哪里？会同步吗？

**数据只保存在你自己设备的浏览器里（localStorage），不会上传到 GitHub，
不同设备、不同人之间不会自动同步。**

- 这个 repo 只托管 APP 程序本身，任何人打开网址用的都是同一个程序，但各记各的账，互相看不到；
- 同一台手机上，主屏幕 APP 的数据会一直保留（除非删掉 APP 或清除 Safari 网站数据）；
- **换手机 / 想把账转到电脑**：在旧设备「数据」页 → 导出收支流水 CSV → 传到新设备 → 「数据」页导入即可，账目一条不少；
- **强烈建议每周导出一次 CSV 备份**（存到微信文件传输助手或 iCloud 都行）。

> 需要多设备实时同步（比如夫妻俩两台手机记同一本账）？那需要加云端后端，
> 欢迎提 [Issue](../../issues)。

## 日常记账流程（收摊 3 分钟）

1. 卖一份点一下品类 → 确认；忙时可收摊后一次补记（把份数改成当天总数）；
2. 进货日（2~3 天一次）在「进货支出」页点快捷按钮，填斤数；
3. 报表页看当天和当月盈亏。**进货日单日净现金为负属正常，看赚钱与否看整月合计。**

## 本地使用（不需要网络托管）

把 `index.html` 下载到电脑，双击用 Safari/Chrome 打开即可，功能完全一样。
注意：本地打开和网址打开是两个独立的数据空间，迁移用 CSV 导出/导入。

## 技术说明

- 纯 HTML + CSS + 原生 JavaScript，单文件，无任何依赖、无构建步骤；
- 数据存 `localStorage`，键名 `seafood_ledger_v1`；
- CSV 导出带 UTF-8 BOM，Excel 打开中文不乱码。
