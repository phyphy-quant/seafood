# 🦐 海鲜捞汁记账

一个给海鲜摊主用的**单文件记账 Web APP**。零依赖、离线可用、手机电脑都行，
专为「全场 20 元/半斤」的捞汁小海鲜摊设计。

**在线使用：<https://phyphy-quant.github.io/seafood/>**

## 功能

- 🧾 **卖货**：点一下品类就记一笔（默认 1 份 = 半斤 = 20 元），份数、价格可改，支持补记往日
- 📦 **进货支出**：快捷按钮一键带入常用品类与批发单价，只需填数量；单位下拉可选（斤/件/袋/盒…）也可自定义；摊位费、燃气、打包盒等支出分类记录
- 📊 **报表**：今日/本月收入、支出、净利，品类销量排行，近 7 天走势，每日汇总表
- 💾 **CSV 导出/导入**：流水、每日汇总、进货记录三种表格，Numbers / Excel / WPS 直接打开
- 📴 **完全离线**：没网也能记账，不需要注册登录

## 📱 iPhone 安装成 APP

1. 用 **Safari** 打开 <https://phyphy-quant.github.io/seafood/>（一定要 Safari，不能在微信里打开）；
2. 点底部中间的**分享按钮**（方框带向上箭头 ⬆️）；
3. 往下滑，点「**添加到主屏幕**」；
4. 名称保持「海鲜记账」，点右上角「**添加**」。

完成后主屏幕会出现 🦐 图标，点开就是全屏 APP，**之后没网也能用**。

## 💻 Mac 安装到程序坞（Dock）

用 Safari 打开同一个网址 → 菜单栏 **文件 → 添加到程序坞…** → Dock 出现 🦐 图标。

## ⚠️ 数据存在哪里？

平时记的账保存在你自己设备的浏览器里（localStorage），离线可用、不依赖网络。
**不同设备之间不会「实时」自动同步**——想共用一本账，用下面的「云端共享账本」。

## ☁️ 云端共享账本（多设备共用一本账）

本仓库的 [`data/`](data/) 文件夹就是共享账本，存三张 CSV：
`流水.csv`（全部明细，唯一数据源）、`每日汇总.csv`、`进货记录.csv`。

APP 的「数据」页有两个按钮：

- **⬇️ 从云端读取账本**：把 `data/流水.csv` 拉下来替换本机账本（任何人都能读，不用 Token）；
- **⬆️ 保存账本到云端**：把本机账本写回仓库的三张 CSV（需要 Token，见下）。

**两台设备共用的正确姿势：开摊先「读取」，收摊后「保存」。**
「保存」是整本覆盖云端，所以不要两个人同时各记各的再先后保存——后保存的会盖掉先保存的；
轮流记、或分工一人记账一人看报表最稳妥。

**申请 Token（一次就好，1 分钟）**

1. GitHub 网页 → 右上角头像 → **Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token**；
2. Repository access 选 **Only select repositories** → 勾选本仓库；
3. Permissions → **Contents → Read and write**；
4. 生成后复制 `github_pat_…`，粘贴到 APP「数据」页 →「☁️ 云端共享账本」→「设置」→ 保存。
   每台需要「保存」功能的设备都填一次（Token 只存在该设备本机）。

> ⚠️ 本仓库是公开的，`data/` 里的账目数据任何人都能看到。介意的话：
> 另建一个**私有**仓库（如 `seafood-data`，建好 `data/` 文件夹即可），
> 在 APP 的云端「设置」里把仓库名改过去，Token 给私有仓库授权——APP 照常读写，外人看不到。

另外仍建议**每周导出一次 CSV 备份**（存微信文件传输助手或 iCloud 都行），双保险。

## 日常记账流程（收摊 3 分钟）

1. 卖一份点一下品类 → 确认；忙时可收摊后一次补记（把份数改成当天总数）；
2. 进货日（2~3 天一次）在「进货支出」页点快捷按钮，填斤数；
3. 报表页看当天和当月盈亏。**进货日单日净现金为负属正常，看赚钱与否看整月合计。**

## 本地使用（不需要网络托管）

把 `index.html` 下载到电脑，双击用 Safari/Chrome 打开即可，功能完全一样。
注意：本地打开和网址打开是两个独立的数据空间，迁移用 CSV 导出/导入。

## 技术说明

- 纯 HTML + CSS + 原生 JavaScript，单文件，无任何依赖、无构建步骤；
- 数据存 `localStorage`，键名 `seafood_ledger_v1`；
- CSV 导出带 UTF-8 BOM，Excel 打开中文不乱码。
